## 🚀 Featured Products

### 🔹 QalamAI — Arabic AI Writing Platform
AI‑powered Arabic writing platform with 8 specialized modes (novels, essays, scripts, poetry, memoires, social content) powered by GPT‑5.x, DALL·E, Whisper, XTTS, and a full publishing/export pipeline.  
**Stack:** React, TypeScript, Tailwind, Shadcn, Radix, Vite, Node.js, Express, PostgreSQL, Drizzle ORM, OpenAI, DALL·E, Whisper, FFmpeg, Coqui XTTS, Stripe, Nodemailer, pdfkit, archiver, docx, Helmet, Zod, PWA.  
👉 **Repo:** https://github.com/AbdellahiBaba/QalamAi  
👉 **Website:** https://qalamai.net  

---

### 🔹 LIMJIBA — E‑Commerce Management System
Enterprise‑grade ERP + e‑commerce platform with inventory, POS, invoicing, multi‑store support, and integrated payments.  
**Stack:** React, TypeScript, Tailwind, Node.js, Express, PostgreSQL, Stripe, Nodemailer, pdfkit, docx, PWA.  
👉 **Repo:** https://github.com/AbdellahiBaba/LIMJIBA  
👉 **Website:** https://limjiba.com  

---

### 🔹 Abu‑Hashim AI — Arabic LLM Framework
Independent Arabic LLM framework for fine‑tuning, evaluation, and inference with custom pipelines for literary and academic Arabic.  
**Stack:** Python, FastAPI, Flask, PyTorch, HuggingFace, OpenAI, FFmpeg, Docker, Linux, security‑hardened APIs.  
👉 **Repo:** https://github.com/AbdellahiBaba/abu-hashim-ai  

---

### 🔹 AegisAI360 — SOC Platform
Multi‑tenant Security Operations Center (SOC) platform for real‑time threat monitoring, alerting, and security analytics.  
**Stack:** TypeScript, React, Node.js, Express, PostgreSQL, Docker, Nginx, Helmet, Zod, GitHub Actions, security‑first architecture.  
👉 **Repo:** https://github.com/AbdellahiBaba/AegisAI360  
👉 **Website:** https://aegisai360.com
